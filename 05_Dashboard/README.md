# Power BI Dashboards

This folder contains the final Power BI dashboards developed for the Retail Analytics project.

The dashboards transform validated SQL data and Power BI measures into interactive reports that help monitor overall business performance, sales trends, customer behavior, and product returns.

---

# Dashboard Overview

## 1. Executive Dashboard

Provides a high-level overview of the company's overall performance.

Main highlights include:

- Revenue
- Profit
- Orders
- Customers
- Products
- Profit Margin
- Executive KPIs
- Year-over-Year Performance

---

## 2. Sales & Customer Behavior Analytics

Combines sales performance analysis with customer purchasing behavior.

Key analyses include:

### Sales Performance

- Revenue Trend
- Category Performance
- Brand Performance
- Product Performance
- Monthly Sales Analysis

### Customer Behavior

- Customer Segmentation
- Revenue by Gender
- Revenue by Age Group
- Customer Spending
- Average Order Value
- Customer Purchase Patterns

---

## 3. Return Performance Analytics

Focuses on product return activities and their financial impact.

Key analyses include:

- Return Rate
- Returned Orders
- Returned Quantity
- Returned Revenue
- Lost Profit
- Return Trends
- Return Reasons
- Year-over-Year Return Performance

---

# Dashboard Features

The dashboards include several interactive features:

- Dynamic KPI Cards
- Year & Quarter Filters
- Cross-filtering
- Interactive Charts
- Year-over-Year Analysis
- Conditional Formatting

---

# Business Value

The dashboards provide a comprehensive view of retail performance by combining executive KPIs, sales analytics, customer behavior, and return analysis into a single reporting solution, enabling faster and more informed business decision-making.
